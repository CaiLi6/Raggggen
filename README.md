# FinAgent RAG Suite

合体名称：**FinAgent RAG Suite**（FinAgent RAG 套件）

概述
----------------
本仓库整合了两个互补项目：

- `Financial-Agent-Client/`：多 agent 金融/股票分析客户端与演示。
- `MODULAR-RAG-MCP-SERVER/`：模块化 RAG + MCP 服务器，提供检索、LLM 接入与协议层。

目的
----------------
将客户端分析能力与模块化 RAG 服务结合，便于在本地或云端快速搭建金融领域的问答、分析与自动化工作流演示环境。
